# 示例代码

本目录包含混合检索评估工具的使用示例。

## 快速开始

### 基础使用示例

`basic_usage.py` - 展示Hybrid Precision指标的基本使用方法：

```bash
python examples/basic_usage.py
```

输出示例：
```
=== 混合检索评估结果 ===
Hybrid Precision: 0.7564
信息熵置信度: 0.8234
互信息置信度: 0.6892
统计显著性: 0.9123
自适应权重 - 稠密: 0.682, 稀疏: 0.318
不确定性惩罚: 0.0456
```

### 简单评估示例

`simple_evaluation.py` - 展示如何使用RAGAS扩展进行完整评估：

```bash
python examples/simple_evaluation.py
```

这个示例会：
1. 执行混合检索评估
2. 生成详细的评估报告
3. 提供优化建议
4. 对比单一检索与混合检索的性能

## 运行要求

确保已安装项目依赖：

```bash
pip install -e .
```

## 自定义使用

您可以根据自己的检索系统修改示例代码：

1. 替换模拟的分数为您的实际检索器输出
2. 修改查询文本和上下文内容
3. 调整评估参数和阈值
